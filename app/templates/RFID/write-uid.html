Write UID to HF RFID CUID Card

RFID UID Writer

This tool allows you to write a new UID to a CUID (Chinese UID) card. CUID cards have writable Block 0, making them useful for cloning access cards.

Note: Only works with CUID cards. Regular Mifare cards have read-only UIDs that cannot be changed.
